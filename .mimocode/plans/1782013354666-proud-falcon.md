```
AGENTS.md
```

## 2. Release 更新日志硬编码

**文件**: `.github/workflows/build.yml`

第 64 行硬编码了 `'桌面记事本 v0.2.0 - 功能增强版'`，需要改成有意义的更新日志。

**改为：**
```yaml
releaseBody: |
  ## 桌面记事本 ${{ github.ref_name }}

  ### ✨ 新增功能
  - 🔒 口令锁屏：防止他人窥视，SHA-256 加密存储
  - ☁️ WebDAV 同步：跨设备数据同步，支持备份/恢复
  - 📊 年度热力图：GitHub 风格，一眼看到全年产出
  - 💻 Mac 风格代码块：红黄绿圆点 + 语言标签
  - 🎨 毛玻璃效果：仪表盘欢迎卡片视觉优化

  ### 🐛 问题修复
  - 移除登录系统，自动创建默认用户
  - 统一 UI 设计系统，修复暗色模式
  - CSP 白名单新增 hitokoto.cn 和 tavily.com

  ### 📦 下载
  | 平台 | 文件 |
  |------|------|
  | Windows | `.msi` 或 `.exe` 安装包 |
  | macOS | `.dmg`（支持 Intel 和 Apple Silicon） |
  | Linux | `.deb` 安装包 |
```

注意：每次发布新版本时，需要手动更新 `releaseBody` 中的更新日志内容。

## 验证

- 提交后检查 `.gitignore` 是否生效
- 推送新 tag 后检查 release 页面是否显示完整更新日志
